Documentación de macros y definiciones en archetype_fits.h.

Autoría: Pablo Gómez y Emmanuel Velásquez

Sintesis de uso:
-----------------------
- Esta implementación ofrece herramientas que simplifican el desarrollo, 
  especialmente para realizar operaciones matemáticas básicas, 
  gestionar mensajes y errores, e implementar bucles de repetición y depuración.
- Busca reducir el uso de memoria al definir tipos de variable facil de interpretar.

Archivo archetype_fits.h:
-------------------------
- Define macros y tipos para operaciones comunes y para facilitar el control de flujo en el proyecto.

Macros y sus funciones:
-----------------------
1. Max(a, b):
    - Calcula el máximo entre 'a' y 'b'.
2. Min(a, b):
    - Calcula el mínimo entre 'a' y 'b'.
3. Frac(a, b):
    - Calcula la fracción de 'a' entre 'b', haciendo casting a tipo Decimal.
4. Loop:
    - Ejecuta un bucle infinito.
5. Msg(str):
    - Imprime un mensaje en la consola seguido de un salto de línea.
6. Failure(cause):
    - Imprime un mensaje de error y termina la ejecución del programa.
7. Repeat(n):
    - Ejecuta un bucle 'for' que itera 'n' veces.
8. Debug(str):
    - Imprime un mensaje de depuración si la variable 'debugMode' está activada.
    - SIEMPRE que se haga uso de esta función, añadir "#define debugMode X" en el
      código, reemplazando X por 0 o 1 para desactivar o activar el modo Debug.
9. End:
    - Finaliza el programa y retorna 0.

Definiciones de tipos personalizados:
-------------------------------------
1. Function:
    - Define un puntero 'void' a funciones sin retorno.
2. ElemType:
    - Define un puntero 'void' para apuntar a tipos vacios.
3. Natural:
    - Alias de tipo para 'unsigned short'
    - Tamaño: 2 bytes
4. Integer:
    - Alias de tipo para 'short'
    - Tamaño: 2 bytes
5. Natural2:
    - Alias de tipo para 'unsigned int'
    - Tamaño: 4 bytes
6. Integer2:
    - Alias de tipo para 'int'
    - Tamaño: 4 bytes
7. Decimal:
    - Alias de tipo para 'double'
    - Tamaño: 8 bytes
8. CharFixed:
    - Alias de tipo para 'const char*'
    - Tamaño dinámico, usado para almacenar strings de texto como una variable.
    - Uso recomendado para nombres de archivo, carpetas, URL o mensajes.
